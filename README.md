Identificação e Contagem de Impurezas em Amostras de GrãosEste projeto propõe o desenvolvimento de um sistema automatizado baseado em visão computacional para identificar e contar grãos pretos (impurezas) em meio a grãos brancos. Utilizando a arquitetura de aprendizado profundo Mask R-CNN, o modelo realiza a segmentação de instância, permitindo identificar e isolar objetos mesmo em situações de sobreposição física.  O objetivo central é modernizar o processo de classificação agrícola, aumentando a precisão, reduzindo erros humanos e tornando o controle de qualidade mais eficiente e confiável.  

🧠 Arquitetura e Inteligência ArtificialModelo: Mask R-CNN utilizando uma rede ResNet-50 combinada a uma Feature Pyramid Network (FPN) como extrator de características.  Abordagem: Segmentação de Instância, capaz de diferenciar contornos individuais de cada grão, superando limitações de métodos clássicos de limiarização de cor que falham ao agrupar grãos colados.  Dataset: Composto por imagens digitais de alta resolução coletadas em ambiente iluminado, expandidas através de técnicas de Data Augmentation (rotações, espelhamento e ajustes de contraste) totalizando 1.200 imagens. Dividido em 70% para treino, 15% para validação e 15% para teste.  

📊 Resultados AlcançadosO desempenho do modelo foi avaliado utilizando métricas consolidadas de detecção de objetos, alcançando excelente performance no conjunto de teste:  mAP@0.5 Global: 0,951   mAP@0.5 Grãos Pretos (Impurezas): 0,961   mAP@0.5 Grãos Brancos (Sadios): 0,942   F1-Score Global: 0,938   A alta performance na detecção das impurezas é crítica para o controle de qualidade, evitando a penalidade comercial por falsos negativos. O ramo de máscaras demonstrou sucesso na individualização dos contornos, mesmo em cenários de alta densidade.  🛠️ Tecnologias UtilizadasLinguagem: Python   Frameworks de IA: PyTorch e Torchvision   Manipulação de Dados e Imagens: NumPy, Pandas, PIL (Pillow)   Métricas e Divisão de Dados: Scikit-learn, pycocotools   Aceleração: Suporte a processamento em GPU NVIDIA via CUDA   

📁 Estrutura do RepositórioO código-fonte foi modularizado para facilitar o treinamento, a validação e a inferência do modelo:converter_coco.py: Script responsável por processar as anotações poligonais das imagens e convertê-las para o padrão COCO (annotations.json), formatando bounding boxes e máscaras.dataset.py / src/graos_dataset.py: Classe personalizada do PyTorch (GraosDataset) para carregamento iterativo das imagens e de suas respectivas máscaras/anotações.predict.py: Módulo de inferência que carrega os pesos do modelo treinado (maskrcnn_graos.pth) e processa novas imagens para gerar as predições.utils.py: Funções auxiliares, como o contador automatizado de classes (sadios vs. impurezas) a partir dos labels detectados.teste_*.py: Conjunto de scripts de validação:teste_dataset.py e testar_coco.py: Validam a integridade do dataset e a leitura do formato COCO.teste_modelo.py: Simula um forward pass de um batch no modelo para verificação das perdas (losses).teste_predict.py: Executa o pipeline completo de detecção, filtrando resultados por grau de confiança (limiar de 0.7) e imprimindo a contagem final no terminal.teste_visualizar.py: Renderiza as máscaras e bounding boxes sobre as imagens originais para conferência visual.

🚀 Como Executar
1. Clone o repositório:Bashgit clone https://github.com/seu-usuario/seu-repositorio.git
cd seu-repositorio

2. Instale as dependências:Certifique-se de ter o Python 3.8+ instalado e instale os pacotes necessários:Bashpip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
pip install Pillow pycocotools scikit-learn pandas matplotlib
(Nota: Ajuste a versão do CUDA na URL de acordo com o seu hardware gráfico).

3. Preparação dos Dados:Coloque as imagens brutas e seus arquivos JSON de anotação na pasta dataset/imagens e execute o conversor:Bashpython converter_coco.py

4. Inferência (Predição):Para testar o modelo em uma imagem específica, certifique-se de que os pesos do modelo estejam salvos em modelos/maskrcnn_graos.pth e execute:Bashpython teste_predict.py
